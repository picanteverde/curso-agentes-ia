# Curso Introductorio: Inteligencia Artificial y Agentes

Un curso completo y gratuito para principiantes que quieren entender y usar agentes de IA.

## Contenido del Curso

| Módulo | Tema | Descripción |
|--------|------|-------------|
| 1 | Introducción a la IA | Conceptos básicos, historia y aplicaciones |
| 2 | Modelos de Lenguaje (LLMs) | Cómo funcionan, capacidades y limitaciones |
| 3 | ¿Qué son los Agentes? | Diferencia con chatbots, componentes clave |
| 4 | Herramientas y Capacidades | Búsqueda, archivos, código, APIs |
| 5 | Patrones y Arquitecturas | Agente único, multi-agente, memoria |
| 6 | Prompts y Comunicación | Técnicas de prompting efectivo |
| 7 | Plataformas para Crear Agentes | Sin código, poco código y para desarrolladores |
| 8 | Casos de Uso Prácticos | Investigación, programación, automatización |
| 9 | Ética y Seguridad | Sesgos, privacidad, uso responsable |
| 10 | Próximos Pasos | Ruta de aprendizaje y recursos |

## Para quién es este curso

- Principiantes sin experiencia previa en programación ni IA
- Personas curiosas sobre cómo funcionan los agentes de IA
- Profesionales que quieren entender cómo aprovechar estas herramientas

## Cómo usar este curso

1. Lee el archivo [curso-agentes-ia.md](curso-agentes-ia.md) de principio a fin
2. Practica los conceptos creando una cuenta en ChatGPT o Claude
3. Experimenta con los proyectos sugeridos en el Módulo 10

## Requisitos previos

Ninguno. El curso está diseñado para principiantes totales.
